# Flex

The `Flex` component renders a responsive flexbox layout component. Extends: `Box`.

<!-- STORY -->

### Usage
```jsx
import { Flex, Box } from 'rimble-ui'
```

<!-- component example here -->
```jsx
<Flex>
  <Box
    p={3}
    width={1/2}
    color='salmon'
    bg='black'>
    Flex
  </Box>
  <Box
    p={3}
    width={1/2}
    color='white'
    bg='salmon'>
    Box
  </Box>
</Flex>
```
