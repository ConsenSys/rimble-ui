# Image

The `Image` component renders the `<img>` HTML element. Inherits props from the `Box` component.
<!-- STORY -->

### Usage
```jsx
import { Image } from 'rimble-ui'
```

<!-- component example here -->
```jsx
<Image
  width={[ 1, 1, 1/2 ]}
  src='https://source.unsplash.com/random/1280x720'
  borderRadius={3}
/>
```
